# Haifa Official Store

## Product Overview
A premium Indonesian Muslim fashion e-commerce website selling hijab, gamis, and modest wear with a Palestine solidarity series.

## App Name
Haifa Official Store

## Tagline
Anggun dalam Iman, Indah dalam Hijab

## Sub-tagline (Arabic)
الحجاب نور، والنور جمال

## Target Users
- Indonesian Muslim women seeking premium modest fashion
- Customers who value quality modest wear with a cause (Palestine solidarity)
- Mobile-first shoppers (Indonesian market)

## Brand Identity
- Premium, elegant, faith-driven modest fashion brand
- Strong Palestine solidarity element (10% donation from Palestine Series)
- Local Indonesian brand with global aesthetic sensibilities
- Tone: Warm, elegant, empowering, spiritually resonant

## Key Sections / Features

### Storefront (Public)
- Homepage (Beranda): Hero, Marquee, Collections, Featured Products, Editorial Banner, Palestine × Haifa, Stats, Newsletter, Footer
- Shop (/shop): Product grid with sidebar filters
- Product Detail (/product/[slug]): Gallery, variants, add to cart, WhatsApp order
- Cart (/cart) and Checkout (/checkout)
- Other: /about, /lookbook, /blog, /contact, /track

### Admin Panel (/admin - private)
- /admin/dashboard: Revenue charts, order stats, top products
- /admin/products: CRUD with multi-photo, variants, badges
- /admin/orders: Status management, shipping input
- /admin/blog: Rich text editor, slug, meta
- /admin/promos: Vouchers, flash sales, discount rules
- /admin/homepage: Drag-and-drop section editor
- /admin/settings: Store config, payment, shipping, WA Business

### Additional Features
- WhatsApp Order Integration (auto-message)
- Flash Sale countdown timer
- Wishlist (localStorage + Supabase sync)
- Product Reviews (star rating + photo)
- Size Guide Modal
- Stock Notification (sold-out email capture)
- Recently Viewed products
- Affiliate/Referral links
- SEO: meta tags, sitemap, OG images

## Tech Stack
- React + Tailwind CSS
- Supabase (auth, database, storage)
- Recharts (dashboard charts)

## Anti-references
- Not a generic white-label store
- Not overly minimalist or sterile
- Not cluttered or discount-store feeling
